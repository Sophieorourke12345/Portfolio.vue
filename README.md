# Sophie O'Rourke Portfolio

Personal portfolio website built with Vue.js to showcase my education, project work, technical skills, experience, achievements, and contact details.

Live site:
https://sophieorourke12345.github.io/Portfolio.vue/

## Features

- Responsive Vue.js portfolio layout
- Dark mode toggle
- Project cards and project detail pages
- Skills, education, experience, and contact sections
- GitHub Pages deployment through GitHub Actions

## Technologies

- Vue.js
- JavaScript
- HTML
- CSS
- Bootstrap
- GitHub Pages

## Local Development

```bash
npm install --legacy-peer-deps
npm run serve
```

## Deployment

Updates are deployed after changes are committed and pushed to the `main` branch.
